The Screen Reader Experience


Imagine you are on a retail site...

You want to purchase a button down shirt. But first you need to know some information about the item.

You want to know the avalible sizes, colors and of course the price.

You will explore this website using a screen reader simulator. You will not have access to the visual design of the site.




How does this work?

You will navigate this site using the HTML tags: Landmarks, Header, Links, Line.

Navigating by Landmarks

Landmarks are divs that break up the main contents of the page. Popular landmarks include a banner, navigation, main and contentinfo for a footer. You can navigate by going to the next or previous landmark.
