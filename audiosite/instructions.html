The Screen Reader Experience


Imagine you are on a retail site...

First you would view the selection of the items.

Then you would find an item of interest and view specifics: possible sizes, colors and styles.

Add the item to the shopping cart and complete a purchase.


Basic Screen Reader Navigation

Landmarks are containers that break up the main contents of the page. Popular landmarks include a banner, navigation and main.

Headers are text labeled as 'headers' on the page.

Links lead the user to another site or different area of the page.

Lines allows to user to navigate the page by every item in order of the HTML code.
