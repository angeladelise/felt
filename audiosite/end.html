Importance of Inclusive Design

39 Million people in the world are blind. Individuals that are blind rely on screen readers to navigate through digital spaces. If a website is not accessible with this assistive technologies, it hinders the ability of this group to access information.

There have been 244 accessibility lawsuits from January 2015 to October 2016; 148 of them have been in the retail space.

The National Federation of the Blind sued Target in 2006 and the case was settled in 2008 when Target agreed to pay $6 million.

Accessible and Inaccessible Web Designs

An image or video without an "alt" tag will make the screen reader read the file name of the item. Non-descriptive file labeling leaves the user completely unaware about the content of the item.

For example, when on www.Amazon.com the screen reader states, “2593218011._ CB308835085_ SR120,50_ .jpg” when it is on the Hanes logo. This long slew of letters and numbers contain no information about the image. This coding flaw prevents screen reader users from understanding this information.


Section 508 Guidelines

Section 508 of the accessibility guidelines provides information about tagging for screen reader interaction.

Every non-text element needs to contain a text alternative (e.g., via "alt", "longdesc", or in element content).

Electronic forms must be accessible with assistive technologies. Users must be able to access information, input field data and submit the form.

All multimedia production, including video must include an audio description.

A variety of color and contrast settings should be provided for the user to control.
